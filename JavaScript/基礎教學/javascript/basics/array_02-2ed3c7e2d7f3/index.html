【 JavaScript 】陣列方法 splice()、slice() 和 split() 使用教學

在 JavaScript 中，splice()、slice() 和 split() 是常用的陣列和字串方法，用於執行不同的操作。下面將詳細介紹這三個方法的用法、差別和範例程式碼，幫助你更好地理解它們的功能和應用。

splice() 方法

splice() 方法用於在陣列中添加、刪除或替換元素。它可以修改原始陣列並返回被刪除的元素。splice() 方法接受三個參數起始索引、刪除的元素個數和可選的新元素。

slice() 方法

slice() 方法用於從陣列中選取指定範圍的元素並返回一個新陣列，而不修改原始陣列。它接受兩個參數起始索引和結束索引（不包含結束索引本身）。

split() 方法

split() 方法用於將字串拆分為子字串，並將結果存儲在一個新陣列中。它接受一個分隔符作為參數，用於指定拆分子字串的位置。

這個比較表格提供了 splice()、slice() 和 split() 三個方法的主要特點。請注意，每個方法都有不同的用途和行為。splice() 用於修改原始陣列、刪除、插入和替換元素，slice() 用於選取指定範圍的元素而不修改原始陣列，而 split() 用於將字串拆分為子字串並返回新的陣列。

總結

在本文中，我們詳細介紹了 JavaScript 中的 splice()、slice() 和 split() 方法的用法和功能。這些方法提供了對陣列和字串進行操作的便利性，並且能夠根據特定需求添加、刪除、替換、選取或拆分元素。藉著熟悉這些方法，你能夠更有效地處理和操作資料。
